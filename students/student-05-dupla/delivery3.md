#### Justification
A escolha técnica foca na mobilidade e colaboração. A plataforma Mobile (iOS) permite o registro estável e seguro de dados diretamente no campo. O desenvolvimento via frameworks modernos garante uma interface intuitiva e de fácil manutenção. O sistema é distribuído para permitir a sincronização em tempo real entre fazendeiros e veterinários através de um banco de dados centralizado, garantindo um controle sanitário compartilhado.
