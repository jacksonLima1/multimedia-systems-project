## User interaction description

Once opened, the software will scan the hardware in the machine where it's installed, and display all information (motherboard, graphic cards, ram, storage device, etc.)
The user can select an item to view more detailed information about the hardware, such as installed and relevant drives, expected lifespan, and compatibility with the other items
The user can also search for items and build an example of a computer with the searched pieces. The application will show it's relevant information.
