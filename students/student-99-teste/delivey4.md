## System Architecture
O sistema smartFrete possui arquitetura web distribuída, baseada em cliente-servidor.

A estrutura funciona da seguinte forma:

- O usuário acessa o sistema pelo navegador web;
- O front-end envia requisições ao servidor;
- O back-end processa as informações;
- Os dados são armazenados no banco de dados;
- O sistema retorna informações atualizadas em tempo real.

Fluxo básico:

Usuário → Interface Web → Servidor → Banco de Dados

## Media Processing
O sistema utiliza principalmente texto e imagens.

- Os textos são utilizados para exibir informações das entregas, rotas e usuários;
- As imagens podem ser usadas para ícones, mapas e identificação visual;
- Os dados são processados pelo servidor e exibidos dinamicamente na interface web;
- O sistema não utiliza processamento de áudio ou vídeo.
